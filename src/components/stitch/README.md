# Componentes importados desde Stitch

Diseño de Stitch: https://stitch.withgoogle.com/projects/14747592987014249993

## Vista integrada

El dashboard de Stitch está disponible en **/stitch** (requiere login).

- Acceso: [http://localhost:3000/stitch](http://localhost:3000/stitch)
- Enlace desde el dashboard: "Vista móvil (Stitch)" en accesos rápidos

## Diseño

- Tema oscuro con glass-morphism
- Navegación inferior (mobile-first)
- Tarjetas: Deuda Total, Ganancia Realizada, Asignaciones Pendientes
- Gráfico de tendencia BCV
- Distribución de estatus (gráfico circular)
- Registro de actividad

## Navegación

Los enlaces del bottom nav apuntan a:
- Inicio → /dashboard
- Ops → /dashboard/operations
- Reportes → /dashboard/reports
